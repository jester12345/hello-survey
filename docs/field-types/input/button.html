Usage:

This page describes the button input type. Generally you want to use an input of type "submit" to submit the form, specifically in the case that the form only has a single submit button. However, if you need to have branching logic in the form to go to a follow-on page that is variable depending on which button was selected, then buttons are a possible choice, as you can assign onClick behaviours to them. However, they do NOT submit the form by default. You must provide an onClick event to the button to make things happen, and it's your responsibility to walk the form and gather the data for taking the next action. So, use with caution, but it may be exactly what you need.
